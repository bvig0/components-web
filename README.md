# 🌐 Components Web

Uma biblioteca prática e leve de componentes front-end desenvolvidos exclusivamente com tecnologias nativas (**HTML5, CSS3 e JavaScript**). O foco é oferecer uma coleção de elementos de UI prontos para uso: visualize, copie o código e aplique no seu projeto.

---

## 📁 Estrutura do Projeto

A organização dos arquivos segue o padrão essencial para uma integração rápida e organizada:

```text
.
├── css/                # Folhas de estilo (estilização dos componentes)
├── js/                 # Scripts (lógica e interatividade nativa)
├── img/                # Assets visuais e ícones
├── pages/              # Demonstrações individuais de cada componente
└── index.html          # Página principal / Vitrine do projeto
```

---

## 🚀 Como utilizar

Este repositório foi desenhado para ser "plug-and-play", sem necessidade de instalações complexas:

**Explore:**  
Acesse o link da demonstração ou navegue pela pasta `pages/`.

**Clone o repositório:**

```bash
git clone https://github.com/bvig0/components-web.git
```

**Copie e Cole:**  
Escolha o componente desejado, copie a estrutura HTML e certifique-se de vincular os arquivos correspondentes das pastas `/css` e `/js` no seu projeto.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e acessível  
- **CSS3**: Layouts modernos (Flexbox/Grid) e responsividade  
- **JavaScript**: Interatividade nativa, sem dependências ou frameworks externos
